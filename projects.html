Portfolio Projects

Here are a few projects I have completed, as well as those I am currently working on, utilizing various programming languages and tools. Feel free to click on the project's picture to be taken to my code on GitHub.

Connect 4

Java

The Connect 4 Game project involved creating a Java application that implemented error handling to manage invalid moves, ensuring a smooth gaming experience. A robust object-oriented implementation of the classic Connect 4 game was developed, complemented by a user-friendly graphical interface that allowed players to interact with game pieces seamlessly. Visual effects were added to enhance the overall user experience. Thorough testing and debugging were conducted to ensure the game's reliability and performance.

Travel Estimator

Python

The project involved creating a sophisticated Python application tailored for backpackers to efficiently estimate trip expenses. A user-friendly graphical interface was developed to enhance interaction, ensuring a seamless experience. Additionally, thorough error handling mechanisms and user-friendly features were integrated to ensure precise and reliable input processing, thereby enhancing the overall functionality and usability of the application.

The Fig Tree Plant Shop

Html, Css, JavaScript

The Plant Shop Website project involved creating a dynamic website to showcase a variety of plant products along with their prices. A fully responsive layout was designed and implemented to ensure optimal viewing on different devices. Interactive elements such as hover effects and modals were added to enhance user engagement. The website was developed with a user-friendly interface and smooth navigation, aimed at providing an enhanced user experience.

Meal-Prep Generator

PHP, SQL

The PHP project is a responsive website designed to provide users with personalized nutritious meal-prep ideas. It includes a secure login system, allowing users to create accounts and access customized meal plans tailored to their dietary preferences, nutritional goals, and specific requirements. The website adapts to different screen sizes, ensuring a seamless experience on desktops, tablets, and mobile devices. Through the user-friendly interface, users can explore a variety of healthy meal options and receive suggestions that align with their individual health objectives (In Progress).
